# Дополнительные поля товара и экспорт в XLSX/PDF

Начиная с версии 0.4.0 сервис подтягивает из прайса и прокидывает до API/экспорта дополнительные атрибуты товара:

* `grapes` — **сортовой состав** (как в колонке «Сорт винограда» в Excel-прайсе).
* `vintage` — **год урожая** (целое число или `null`, если год не указан).
* `vivino_url` — **рейтинг Vivino** (значение из колонки «Vivino» в прайсе, например `3.9`, `4.4` и т.п.).
* `vivino_rating` — **экспертный рейтинг** (значение из колонки «Рейтинг» в прайсе, например `96`, `93`, `94`).
* `supplier` — **поставщик/производитель** (колонка «Поставщик»).
* `image_url` — **фото бутылки / этикетки**.
* `producer_site` — **сайт производителя** (если указан в прайсе).

Эти поля доступны:

* в REST-поиске `GET /api/v1/products/search`,
* в JSON-экспорте поиска `GET /export/search?format=json`,
* в экспорте поиска в **XLSX** и **PDF**,
* в карточке SKU (`GET /export/sku/<code>?format=json|pdf`, доступ по API-ключу).

## Формат экспорта поиска

### JSON

```http
GET /export/search?format=json&limit=5
```

Ответ:

```json
{
  "Count": 5,
  "value": [
    {
      "code": "D011194",
      "title_ru": "Altesino Brunello di Montalcino Montosoli ...",
      "price_list_rub": 33276,
      "price_final_rub": 33276,
      "color": "красное",
      "region": "Toscana",
      "producer": "Altesino",
      "grapes": "Санджовезе 100%",
      "vintage": 2017,
      "vivino_url": "4.4",
      "vivino_rating": 96,
      "supplier": "Altesino"
      // ...другие поля
    }
  ]
}
```

### XLSX / PDF

```http
GET /export/search?format=xlsx&limit=100&in_stock=true
GET /export/search?format=pdf&limit=100&in_stock=true
```

В отчётах используется расширенный набор колонок (по порядку):

1. `Код`
2. `Название`
3. `Цена прайс`
4. `Цена финальная`
5. `Цвет`
6. `Регион`
7. `Производитель`
8. `Сортовой состав`
9. `Год урожая`
10. `Рейтинг Vivino`
11. `Экспертный рейтинг`
12. `Поставщик`

Таким образом, Excel/PDF-отчёты можно сразу использовать для дегустаций, презентаций и внутренних отчётов — без ручного дописывания сортового состава, года урожая и рейтингов.
